Comment utiliser le minuteur

Pour activer le minuteur et voir les champs :

  • Cliquez sur le bloc rose (minuteur rouge). Le curseur se transformera en pointer pour indiquer qu’il est interactif.
  • Les champs de minute et seconde apparaîtront et deviendront modifiables.
  • Appuyez sur le bouton Refresh deux fois pour lancer le minuteur :
    • La première fois, il se met en pause et prépare le compte à rebours.
    • La deuxième fois, il démarre correctement.
  • Important : ne cliquez pas trop rapidement sur Refresh, cela pourrait provoquer une boucle infinie.

Le minuteur affichera le temps restant sous la forme mm:ss dans le bloc noir central.
